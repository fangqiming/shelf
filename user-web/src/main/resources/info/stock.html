/trade/search_gain

查找最新的推荐信息获利情况 此接口同样适用于盈利历史

/trade/search_gain get

get

get /trade/search_gain

  • Request
  • Response

Query Parameters

  • pageSize: required (integer)

    页大小

  • pageNo: required (integer)

    第几页

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
   "code": 0,
   "data": {
     "entities": [
       {
         "profit": 15,             //收益率之和
         "sellNumber": 1,          //卖出的股票份数
         "date": "2017-04-04"      //日期
       },
       {
         "profit": 13,
         "sellNumber": 2,
         "date": "2017-04-03" //实际盈利为 profit / sellNumber / 100  取两位小数
       }
     ],
     "total": 2
   }
 }
