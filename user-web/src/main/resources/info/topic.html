/topic/create

创建话题

/topic/create post

post

post /topic/create

  • Request
  • Response

Body

Media type: application/json

Type: object

Example:

{
	"content":"文章内容",
	"title":"测试文章",
	"userCode":"测试码"
}

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
   "code": 0,    //错误码
   "data": {
     "id": 4     //文章主键
   }
 }

/topic/search

分页查找话题

/topic/search get

get

get /topic/search

  • Request
  • Response

Query Parameters

  • pageSize: required (integer)

    页大小

  • pageNo: required (integer)

    第几页

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
  "code": 0,
  "data": {
    "entities": [
      {
        "id": 4,               //文章主键
        "content": "文章内容",  //文章内容
        "title": "测试文章",    //文章标题
        "userCode": "测试码",   //测试码
        "clickNum": null,       //点击数
        "createdTime": 1524626318000 //创建时间 毫秒数
      },
      {
        "id": 3,
        "content": "文章内容",
        "title": "测试文章",
        "userCode": "测试码",
        "clickNum": null,
        "createdTime": 1524626315000
      }
    ],
    "total": 3
  }
}

/reply/createFirst

/reply/createFirst post

post

post /reply/createFirst

  • Request
  • Response

Body

Media type: application/json

Type: object

Example:

{
	"topicId":1,       //话题主键
	"userCode":"123",  //用户码
	"content":"Hello World"    //评论内容
}

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
   "code": 0,
   "data": {
     "id": 8   //评论的主键
   }
 }

/reply/createReply

/reply/createReply post

post

post /reply/createReply

  • Request
  • Response

Body

Media type: application/json

Type: object

Example:

{
	"userCode":"123",          //用户码
	"content":"我是评论内容",    //评论内容
	"replyId":1                //被评论的id
}

Body

Media type: application/json

Type: object

Example:

{
  "code": 0,    //错误码
  "data": {
    "id": 10    //该评论的主键
  }
}
